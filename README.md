# XMODITS Core Library

Work in progress

## Supported Formats
| Extension | Format |
| --- | --- |
| IT | Impulse Tracker |
| XM | Extended Module | 
| S3M | Scream Tracker 3 |
| MOD | Amiga ProTracker |
| MPTM | ModPlug Tracker module (Impulse Tracker) |
| UMX | Unreal Music Package (Containing above) |
